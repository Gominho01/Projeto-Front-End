Troca de Senha de Clientes

Instruções de senha:

A senha deve ter pelo menos 6 caracteres, contendo pelo menos:

  • Um caractere numérico
  • Uma letra maiúscula
  • Um dos caracteres especiais: @ # $ % & * ! ? / \ | - _ + . =

Os seguintes caracteres não são permitidos: ¨ { } [ ] ́ ` ~ ^ : ; < > , “ ‘
